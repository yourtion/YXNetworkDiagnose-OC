Network Diagnose Framework for iOS and OSX.

## How to use

### Install

#### CocoaPods

Add in your `Podfile`:

```
platform :ios, '8.0'

pod 'NetworkDiagnose'
```

Run `pod install`

#### Carthage

```
github "yourtion/YXNetworkDiagnose-OC"
```

### Use

import head file :

```objc
#import <YXNetworkDiagnose/YXNetworkDiagnose.h>
```
